
#### dart的入口

`dart`不像js整个文件都会运行一次,它是需要主入口(主函数)来指定运行的

像是下面的代码:
```dart
class student {}  
  
String getName(params) {  
  return '张三' + params;  
}  
  
void main() {  
  print('test dart');  
  final fun1 = getName('hello');  
  print(fun1);  
  // 不可以用 const 接收一个function  
  // const fun2 = getName('params');  // print(fun2);}
```
