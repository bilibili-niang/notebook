# 成员变量

如在这个里面:

 
private int ages;

 
 
ages叫成员变量或者字段,不叫属性

 
 
在下面这个中ages叫做属性

void setAge(int a){

System.out.println(ages);

}

 
void setAge(int a){

ages=3;

}

int getAge(){

return age;

}

 
加上get,set之后才叫属性
