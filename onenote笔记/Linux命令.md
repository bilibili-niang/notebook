# Linux命令

1.指令与选项

what?

指在Linux终端(命令行)中输入的指令

一个完整的指令的标准格式,Linux通用的格式

所有命令以#开始,中括号表示可写可不写,操作对象可以不写,默认当前对象,

指令[-选项] [操作对象]

1.有个别命令不遵循此格式

2.当有多个选项时,可以写在一起

3.简化选项与完整选项(-a等于--all)

一个指令可以包含多个选项,

操作对象也可以是多个,

 
 
部分简化指令:

-a显示所有文件,包括隐藏文件

-l详细信息显示

-d查看目录属性

在Linux中"~"代表当前用户的家目录
