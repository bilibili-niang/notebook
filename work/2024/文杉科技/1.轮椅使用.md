#### 响应的catch,finally

`封装的有方法`:
```tsx
Modal.confirm({  
  title: '是否确认作废',  
  maskClosable: true,  
  content: (  
    <>  
      确定要作废{' '}  
      <span class="color-warn">  
        {record.merchantName}-{record.productName}-{record.packageName}  
      </span>{' '}  
      吗？  
    </>  
  ),  
  onOk: () => {  
    $deprecatedApps(record.id)  
      .then((res: any) => {  
        if (res.success) {  
          message.success(res.msg || '操作成功')  
        } else {  
          message.info(res.msg)  
        }      })      .catch(useRequestErrorMessage)  
      .finally(() => refresh())  
  }})
```
坐轮椅即可

#### schema中指定某一个properties的类名:
```tsx
const { onCreate: batchOnCreate } = useCrud({  
  title: '场地',  
  schema: () => {  
    return {  
      type: 'object',  
      properties: {
        prefixName: {  
          title: '场地前缀',  
          type: 'string',  
          required: (rootValue: any) => {  
            return !rootValue.postName  
          },  
          config: {  
            class: 'site-prefix-tips',  
            placeholder: '前缀,后缀二选一'  
          },  
          condition(rootValue, parentalValue, parentalProps) {  
            return !rootValue.postName  
          }  
        }
    }    
   }  
  },
})
```

可以看到在上面的`场地前缀`中加的是`class`而不是`className`,这里加`className`之前的类名会被覆盖掉
#### 前台的 usePopup 响应式

```tsx
// 一级的选择  
const showPagePop = () => {  
  pagePop.value = usePopup({  
    placement: 'bottom',  
    content: () => (  
      <div class="show-page-pop">  
        <Tabs  
          size="large"  
          direction="vertical"  
          title-scroll  
          v-model={selectionValue.value}  
          onChange={e => {  
            selectionValue.value = e.paneKey  
            pagePop.value?.update()  
          }}  
        >  
          {  
            tabsValue.map((it, index) => {  
              return <TabPane  
                paneKey={it.value}  
                title={it.label}>  
                {it.label}-{it.value}  
              </TabPane>  
            })  
          }  
        </Tabs>
      </div>    )  
  })  
}
```
把 `content` 传入箭头函数即可
